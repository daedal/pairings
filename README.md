#Pairings

A site which gives suggestions on cheese/wine pairings

####Technology Used:

Back End
- Ruby 2.0  
- Rails 4  
- PG Dataabase 

Front End
- Zurb Foundation  
- CoffeeScript  
- ERB/HAML  
- SCSS  


The bartendr.me team wants to extend thanks to the team at [import.io](http://www.import.io)



To run:
```
$ git clone <link to repo>
```

```
$ cd bartendr
```
```
rake db:create db:migrate db:seed
```
```
rails s
```
