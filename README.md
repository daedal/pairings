#Pairings

A site which gives suggestions on cheese/wine pairings

####Technology Used:

Back End
- Ruby 2.0  
- Rails 4  
- PG Database 

Front End
- Zurb Foundation  
- CoffeeScript  
- ERB/HAML  
- SCSS  


To run:

``` sh
$ git clone <link to repo>

$ cd pairings

$ rake db:create db:migrate db:seed

$ rails s
```
